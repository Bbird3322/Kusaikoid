# Kusaikoid<br>
TTSdemo環境を用いたreadspeakerモデルの音声生成サイトです。
htmlですので、お使いのブラウザから使用してください。ネット環境がないと動作しません。

# 使い方<br>
1.リンクからKusaikoid_betav3.htmlをダウンロード<br>
2.任意のブラウザでHTMLを開く。(オンライン環境のみ動作)<br>
3.入力タブより任意のテキストを入力。<br>
4.ダウンロードボタンでmp3化<br>
